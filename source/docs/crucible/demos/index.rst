Demos
===============================================

Some things are easier to see than to be told. A demo is run rather than played: it puts a consequence on a
screen, synthetic or real, so a room can watch it happen instead of taking it on faith. Being told that
sensor data can be manipulated is one thing. Watching the voltage lines spike while the physical grid is fine
is another.
