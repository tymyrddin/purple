Experiments
===============================================

.. image:: /_static/images/experiments.png

An experiment introduces one controlled change into a running environment and watches what actually
happens, rather than what the documentation predicts. Does the alert fire against this technique. Does the
escalation path answer at two in the morning. Does the procedure hold when someone follows it under load.
The value is in the gap between the expected result and the observed one.
