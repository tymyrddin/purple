Room for TryHackMe
===============================================

TryHackMe is a beginner-friendly platform where each "room" is a complete learning module with multiple tasks,
questions, and a deployable target. A submitter builds a virtual machine locally using VMware or VirtualBox,
exports it to OVA format, and uploads it to TryHackMe. TryHackMe converts the VM for its AWS infrastructure.
